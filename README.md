# Simulador de YouTube 

> Projeto desenvolvido para praticar conceitos de Programação Orientada a Objetos (POO) em Java simulando um ambiente de reprodução de vídeos parecido com a plataforma YouTube

## 💻 Tecnologias utilizadas:  
* Java
* Eclipse IDE
* Git e GitHub

## ⚙ Funcionalidades:
* 🟩 Dar play
* 🟥 Pausar
* 👍 Curtir
* ⚖ Avaliar
